# Linux-Bash-Scripting

## How to run the shell codes

1. Open nano gnu on your Linux Terminal
<img width="815" height="579" alt="Screenshot from 2026-03-10 14-03-33" src="https://github.com/user-attachments/assets/36b13e88-39b8-4d32-9d32-da0909db7e86" />

2. Name your file.
   Ex: nano <file_name.sh>
<img width="815" height="579" alt="image" src="https://github.com/user-attachments/assets/ea6d84cc-d0a9-457d-994d-046281b2c04b" />

3. Your file is created after you click enter as it should look something like this
<img width="815" height="579" alt="image" src="https://github.com/user-attachments/assets/c116ca49-5dd0-4fd5-a889-ab7d6d0b3312" />

4. Save your file content using "ctrl + s" in the nano gnu terminal

5. Type this command to configure the execution of the file contents

``` bash
chmod +x <file_name.sh>
```
6. Run the file using the below command

``` bash
./<file_name.sh>
```
